# Microsserviço de Compra

## Agregado

- Cartão de Crédito
  número
  
- Compra
  loja, valor, data
  
## Regras

- Uma compra pode ter apenas um cartão vinculado
- Cada compra precisa de um número de cartão de crédito válido

## Eventos

- CompraCriada: disparado quando uma nova compra é criada no sistema.

## Comandos

- Criar Compra
